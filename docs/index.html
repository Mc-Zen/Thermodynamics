Theoretische Physik IV - Thermodynamik/Statistische Physik

Hier ist stets die aktuelle Version des LaTeX-Skripts von der Vorlesung Thermodynamik/Statistische Physik (Theoretische Physik IV) im Sommersemester 2022 zu finden. Es wird regelmäßig aktualisiert.

Wenn du einen Fehler findest, eine Anmerkung oder einen Vorschlag hast, dann poste bitte ein Issue auf GitHub. Ich gebe mir zwar Mühe, mache aber Fehler und würde diese gerne beheben. Gerne auch auf Grammatik/Rechtschreibung aufmerksam machen :)

Wer sich mit git auskennt, kann auch das Repository klonen und Pull Requests erstellen.

Die Versionsnummer hat 3 Zahlen, z.B. 1.3.2. Die letzte Zahl wird immer erhöht, wenn kleine Korrekturen gemacht wurden, die vorletzte, wenn neue Kapitel dazugekommen sind. Die vorderste Zahl erhöhe ich nur bei sehr großen Änderungen.
